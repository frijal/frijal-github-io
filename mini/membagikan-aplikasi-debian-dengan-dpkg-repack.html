Membagikan Aplikasi Debian dengan dpkg-repack

Banyak pengguna Debian yang rajin menjaga kebersihan direktori /var/cache/apt/archives/ dengan menjalankan apt-get clean. Itu bagus, tetapi kadang muncul masalah saat ingin berbagi aplikasi dengan teman di lokasi tanpa internet.

Bayangkan Anda ingin memperlihatkan aplikasi baru di laptop, lalu teman Anda juga ingin memasangnya. Tidak ada internet, tidak bawa repo, hanya ada flashdisk kosong. Apa solusinya? Gunakan dpkg-repack!

Langkah-langkah Menggunakan dpkg-repack

  1. Pastikan paket dpkg-repack sudah terpasang. Jika belum, Anda perlu koneksi internet untuk menginstalnya.
  2. Jika ingin menjalankan tanpa sudo, gunakan bantuan fakeroot:
    sudo fakeroot -u dpkg-repack nama-paket
  3. Contoh: Anda ingin berbagi aplikasi pemutar musik Amarok. Jalankan:
    dpkg-repack amarok
  4. File amarok.deb akan muncul di direktori saat ini. Salin ke flashdisk dan bagikan!
