Solusi LibreOffice Lambat di Jaringan

Panduan Cepat Atasi Masalah Saat Buka Dokumen dari Jaringan

Pernah gak sih kamu ngalamin, pas mau buka dokumen LibreOffice yang disimpan di folder jaringan, eh, loading-nya lamaaa banget? Sampai bikin sebel? Ternyata, masalah ini cukup umum, terutama buat pengguna Linux.

@ubuntu_id kenapa ya stlh updt k 14.04 proses LOffice via share jadi lemot, buka file share aja bisa 5 mnit. Bisa bantu solusi?

— Janu S Hastomo (@janutomo) December 1, 2014

Dulu, saya juga pernah merasakan hal yang sama. Untungnya, ada trik sederhana yang bisa langsung ngatasin masalah ini. Ternyata, kelambatan ini sering terjadi karena ada fitur yang namanya **GIO** (Glib Input/Output) yang terlalu cerewet saat mengakses file di jaringan. Solusinya? Nonaktifkan saja fitur itu, dan paksa LibreOffice untuk langsung menggunakan **GVFS-Fuse**.

---

Langkah-langkahnya Gampang Banget!

Ikuti dua langkah simpel ini, dan dijamin LibreOffice-mu bakal ngebut lagi:

1. Pastikan gvfs-fuse Sudah Terpasang

Pertama-tama, pastikan paket gvfs-fuse sudah terpasang di sistemmu. Kalau belum, buka terminal dan ketik perintah ini:

sudo apt-get install gvfs-fuse

2. Nonaktifkan GIO untuk LibreOffice

Setelah itu, salin dan jalankan baris perintah sakti ini di terminal:

sudo sed -i 's/X-GIO-NoFuse=true/#X-GIO-NoFuse=true/' /usr/share/applications/libreoffice*

Perintah ini akan mengedit file konfigurasi LibreOffice dan "menghidupkan" opsi yang menonaktifkan GIO. Jangan lupa, gunakan sudo ya, karena perintah ini butuh hak akses root.
