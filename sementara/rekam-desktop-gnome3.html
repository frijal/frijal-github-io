Rahasia Rekam Desktop di Gnome 3: Tanpa Aplikasi Tambahan!

Siapa sangka, desktop Gnome 3 punya fitur perekam layar tersembunyi yang siap pakai? Gak perlu install apa-apa lagi!

Seringkali kita butuh merekam aktivitas di desktop, entah itu untuk bikin tutorial, ngasih demo, atau sekadar dokumentasi. Nah, buat kamu pengguna Gnome 3, ada kabar gembira! Ternyata desktop kesayangan kita ini sudah punya fitur perekam layar bawaan yang langsung nempel, lho. Kamu gak perlu repot-repot install aplikasi pihak ketiga lagi.

Penasaran gimana cara ngaktifinnya? Gampang banget, cuma butuh satu kombinasi tombol rahasia!

Aktifkan Perekam Layar Seketika!

Untuk memulai merekam layar, kamu cukup menekan kombinasi tombol sakti ini:

Ctrl + Shift + Alt + r

Begitu kamu menekannya, perekaman akan langsung dimulai secara otomatis!

Setelah menekan kombinasi tombol itu, perhatikan bagian kanan bawah layar desktopmu. Akan ada balon kecil berwarna merah yang muncul. Itu tandanya perekaman sedang berjalan! Keren, kan? üòâ

Balon merah di pojok kanan bawah menandakan perekaman aktif. Klik untuk melihat lebih jelas!

Berhenti Merekam & Lokasi File Video

Kalau sudah selesai merekam, kamu cukup menekan kombinasi tombol yang sama lagi:

Ctrl + Shift + Alt + r

Perekaman akan otomatis berhenti dan videomu siap diintip.

Video yang sudah direkam akan tersimpan otomatis di direktori /home/NAMAMU/Videos (atau langsung di /home/NAMAMU tergantung versi Gnome) dengan nama yang formatnya seperti ini: shell-TAHUNBULANTANGGAL-URUTAN.webm. Contohnya: shell-20111105-2.webm. Jadi, file-nya akan berekstensi .WEBM.
