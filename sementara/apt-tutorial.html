APT adalah package manager bawaan sistem operasi Debian dan turunannya (seperti Ubuntu, Linux Mint, dll). Dengan APT, kita bisa dengan mudah mencari, menginstal, menghapus, dan mengelola aplikasi langsung dari terminal.

1. Fungsi Utama APT

  • Menginstal aplikasi dari repository resmi Debian.
  • Menghapus aplikasi yang tidak diperlukan.
  • Melakukan update dan upgrade sistem.
  • Mencari aplikasi berdasarkan nama atau deskripsi.
  • Mengelola dependensi secara otomatis (APT akan memasang paket tambahan yang dibutuhkan oleh aplikasi).

2. Cara Dasar Menggunakan APT

3. Mencari Nama Aplikasi dengan APT

Gunakan perintah berikut untuk mencari aplikasi yang tersedia:

        
        apt search nama-aplikasi

Contoh mencari aplikasi htop:

        
        apt search htop

4. Memasang Aplikasi dengan APT

Gunakan perintah berikut untuk memasang aplikasi:

        
        sudo apt install nama-aplikasi

Contoh memasang htop:

        
        sudo apt install htop

5. Menghapus Aplikasi dengan APT

Jika aplikasi tidak diperlukan, bisa dihapus dengan:

        
        sudo apt remove nama-aplikasi

Untuk menghapus sekaligus file konfigurasinya (penghapusan total):

        
        sudo apt purge nama-aplikasi

Kesimpulan

APT memudahkan kita dalam mengelola aplikasi di Debian dan turunannya. Dengan APT, kita tidak perlu repot mengunduh file aplikasi manual satu per satu, karena semua paket disediakan di repository resmi dan bisa dipasang langsung dari terminal.
