Menyatukan Pecahan Film dengan Terminal (cat command)

Panduan praktis menyatukan potongan film menggunakan perintah cat di Linux.

Sering kali film yang diunduh dari internet terbagi menjadi beberapa bagian kecil. Untuk menyatukannya sebenarnya sangat mudah, hanya membutuhkan terminal dan sedikit perintah.

Contoh Direktori

Langkah Penyatuan

  1. Buka Terminal.
  2. Ketik perintah cat.
  3. Tambahkan semua nama file secara berurutan, dipisahkan dengan spasi.
  4. Gunakan simbol > diikuti nama file output baru.

Contoh Perintah

cat film.part1.avi film.part2.avi film.part3.avi > film-lengkap.avi

Hasilnya adalah satu file film utuh pada folder yang sama, tanpa mengubah kualitas aslinya.

Catatan Penting

Teknik ini hanya untuk menyatukan file pecahan, bukan menggabungkan film multi-CD (misalnya CD1, CD2, dst). Jika menggunakan LibreOffice, metode serupa bisa dipakai untuk dokumen yang terpecah.
