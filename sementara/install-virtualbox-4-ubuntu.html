Install VirtualBox 4 di Ubuntu Jadul

Kangen sama tampilan Ubuntu 10.04 Lucid Lynx atau 11.04 Natty Narwhal? Yuk, kita pasang VirtualBox versi 4 yang cocok buat mereka!

Halo, Sobat Linux! Kadang ada kalanya kita butuh menjalankan sistem operasi lawas untuk testing atau sekadar nostalgia. Nah, VirtualBox adalah salah satu tool virtualisasi andalan. Artikel ini khusus buat kamu yang masih setia atau perlu menjalankan Ubuntu versi lama (sekitar 10.04 sampai 11.04) dan ingin menginstall VirtualBox 4.

Prosesnya gampang banget, cuma butuh tiga langkah utama di terminal. Siapin kopimu, kita mulai!

Tiga Jurus Jitu via Terminal

Langkah 1: Tambahkan Repository VirtualBox

Pertama, kita kasih tau sistem di mana lokasi download VirtualBox. Perintah ini akan otomatis membuat file list repository baru.

echo "deb http://download.virtualbox.org/virtualbox/debian $(lsb_release -sc) contrib" | sudo tee /etc/apt/sources.list.d/virtualbox.list

Langkah 2: Daftarkan Kunci Oracle

Selanjutnya, kita perlu menambahkan kunci publik dari Oracle agar repository yang tadi kita tambahkan dipercaya oleh sistem.

wget -q http://download.virtualbox.org/virtualbox/debian/oracle_vbox.asc -O- | sudo apt-key add -

Langkah 3: Update & Install VirtualBox 4.0

Terakhir, update daftar paket dan langsung install VirtualBox-nya. Kita juga install `guest-utils` biar integrasi antara host dan guest OS lebih mulus.

sudo apt-get update; sudo apt-get install virtualbox-4.0; sudo apt-get install virtualbox-ose-guest-utils

Setelah proses instalasi selesai, kamu bisa menemukan VirtualBox di menu aplikasi. Tinggal jalankan dan mulai buat mesin virtual pertamamu!



Tampilan VirtualBox Manager setelah terinstall. Klik untuk memperbesar.

Untuk info lebih lanjut atau jika kamu butuh versi lain, jangan ragu buat mampir ke situs resminya di sini. Selamat bernostalgia!
