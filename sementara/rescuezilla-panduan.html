üîé Apa itu Rescuezilla?

  • Rescuezilla adalah fork dari Clonezilla dengan antarmuka grafis yang lebih ramah pengguna.
  • Bisa digunakan untuk:
    • Backup & restore partisi atau seluruh disk.
    • Clone disk ke disk lain (semua partisi ikut).
    • Verifikasi image backup.
  • Berbasis Debian/Ubuntu dan tersedia dalam bentuk ISO bootable.

Gambar 1: Tampilan awal Rescuezilla setelah boot

üì• Cara Mendapatkan Rescuezilla

  1. Unduh ISO Rescuezilla terbaru dari rescuezilla.com/download
    Contoh: rescuezilla-2.4.2-64bit.jammy.iso atau 2.5 noble
  2. Gunakan Balena Etcher atau aplikasi sejenis untuk membuat USB bootable.
  3. Boot komputer dari USB tersebut. Rescuezilla siap digunakan!

üíæ Membuat Backup

  1. Klik ikon Backup.
  2. Pilih disk yang ingin dibackup (misal SSD internal).
  3. Pilih partisi yang akan disertakan.
  4. Tentukan lokasi penyimpanan (disk eksternal/network share).
  5. Pilih folder tujuan.
  6. Beri nama file backup (default: tanggal + -img-rescuezilla).
  7. Pilih metode kompresi: gzip, zstandard, bzip2, atau tanpa kompresi.
  8. Klik Next ‚Äì proses backup dimulai!
