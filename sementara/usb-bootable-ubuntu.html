Membuat USB Bootable Ubuntu Tanpa USB Creator

Ubuntu menyediakan USB Creator karena berkas *.ISO Ubuntu tidak menyertakan fitur Hybrid ISO seperti Fedora, OpenSuse, dan beberapa distro lainnya.

Hybrid ISO menyertakan berkas syslinux atau isolinux yang dibuat oleh H. Peter Anvin agar berkas ISO bisa langsung booting tanpa bantuan aplikasi tambahan.

Jika Ubuntu telah mengadopsi Hybrid ISO, untuk membuat FlashDisk bisa booting cukup dengan menyalin berkas ISO menggunakan dd. 

dd if=Berkas-Ubuntu.iso of=/dev/FlashDisk

Jika masih ingin menggunakan USB Creator atau yang serupa, masih diperbolehkan.
