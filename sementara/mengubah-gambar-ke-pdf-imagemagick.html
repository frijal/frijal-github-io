Mengubah Banyak Gambar Menjadi PDF dengan ImageMagick

Bagi rekan-rekan yang gemar mengoleksi komik atau menyimpan dokumen hasil scan, seringkali kita memiliki banyak sekali file gambar yang terpisah. Tentu akan merepotkan jika harus membuka satu per satu. Solusi praktisnya adalah dengan menggabungkan semua gambar menjadi satu file PDF.

Dengan bantuan ImageMagick, proses ini dapat dilakukan dengan cepat. Pastikan Anda sudah memasang paket ImageMagick di sistem Anda.

Cara Menggabungkan Gambar ke PDF

convert *.jpg filebaru.pdf

Perintah di atas akan menggabungkan semua file berformat .jpg di folder Anda menjadi satu file PDF bernama filebaru.pdf. Anda juga bisa mengganti .jpg dengan ekstensi lain seperti .png, .jpeg, atau .bmp.
