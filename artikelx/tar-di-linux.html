Perintah dasar

tar -czf backup.tar.gz folder/

Command ini akan membuat file archive terkompresi dari folder dan semua isinya‚Äîkayak WinRAR tapi langsung dari terminal dan lebih powerful!

Saran: Simpan di lokasi aman dan pakai nama jelas biar gampang dicari nanti.

Variasi berguna

  • Extract
    tar -xzf file.tar.gz
    Extract file .tar.gz ke direktori saat ini.
  • Backup bertanggal
    tar -czf backup-$(date +%Y%m%d).tar.gz /home/user/
    Nama file otomatis pakai tanggal hari ini.
  • Intip isi
    tar -tf archive.tar.gz
    Lihat isi archive tanpa extract.
  • Exclude .tmp
    tar -czf backup.tar.gz --exclude="*.tmp" folder/
    Kompres tapi skip file .tmp.

Parameter penting

  • -c: Create (buat archive)
  • -x: Extract (buka archive)
  • -z: Gunakan gzip compression
  • -f: Specify filename
  • -v: Verbose (tampilkan proses)

Alternatif

  • zip:
    zip -r backup.zip folder/
  • unzip:
    unzip file.zip
  • 7z:
    7z a backup.7z folder/
    Biasanya ukuran kompresi lebih kecil.

Kapan digunakan

  • Backup: Simpan cadangan data penting biar tenang.
  • Transfer: Kirim banyak file sekaligus jadi satu paket.
  • Hemat ruang: Kompres biar storage gak cepat penuh.
  • Distribusi: Rilis software atau project rapi dalam satu archive.
