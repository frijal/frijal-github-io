🔎 Apa itu Flatpak?

Flatpak adalah solusi modern untuk instal aplikasi di hampir semua distro Linux. Dengan Flatpak, kamu nggak perlu pusing soal format paket distro—satu kali install, bisa jalan di Ubuntu, Fedora, Arch, Mint, dan banyak lagi. Flatpak juga punya ribuan aplikasi yang update!

⚙️ Instalasi Flatpak

1. Cek apakah sudah terpasang

flatpak --version

2. Instalasi di Ubuntu

sudo apt install flatpak

Supaya dapet versi terbaru, tambahkan PPA:

sudo add-apt-repository ppa:flatpak/stable -y
sudo apt update
sudo apt install flatpak

3. Fedora

Flatpak biasanya udah terpasang by default.

🚀 Mengaktifkan Flatpak

Setelah instalasi, jalankan:

sudo systemctl start flatpak-system-helper
sudo systemctl status flatpak-system-helper

🌐 Menambahkan Remote (Flathub)

Remote itu “sumber” aplikasi Flatpak. Yang paling hits: Flathub. Kalau ingin dapet aplikasi terbaru, wajib banget nambahin Flathub:

flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
flatpak remotes

📥 Instalasi Aplikasi

Cari aplikasi:

flatpak remote-ls | grep tux

Contoh install game Super Tux Party:

flatpak install flathub party.supertux.supertuxparty

📋 Manajemen Aplikasi

  • Daftar aplikasi: flatpak list
  • Uninstall aplikasi: flatpak uninstall flathub org.opensurge2d.OpenSurge
  • Update semua aplikasi: flatpak update
  • Perbaiki daftar paket: flatpak repair

🔐 Mengatur Izin Aplikasi dengan Flatseal

Flatseal adalah aplikasi GUI buat atur izin Flatpak, dari akses jaringan, printer, GPU, sampai Bluetooth.

flatpak install flathub com.github.tchx84.Flatseal
flatpak run com.github.tchx84.Flatseal

🧹 Menghapus Flatpak dan Aplikasi

Ingin bersih-bersih Flatpak dan aplikasinya?

flatpak uninstall --all --delete-data

Lalu hapus Flatpak sesuai manajer paket distro kamu.

🎯 Kesimpulan

Flatpak bikin instal aplikasi di Linux makin praktis dan universal. Dengan Flathub, kamu bisa dapet ribuan aplikasi update tanpa ribet format paket. Belum lagi, Flatseal bikin atur izin aplikasi semudah klik-klik aja. Dengan Flatpak, dunia Linux makin fleksibel!
