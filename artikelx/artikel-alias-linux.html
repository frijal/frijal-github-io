üîé Apa itu Alias?

Alias adalah cara bikin shortcut perintah di terminal Linux. Jadi, perintah panjang bisa disingkat jadi satu kata aja‚Äîhemat waktu, nggak pegel ngetik!

Contoh alias basic:

alias ls="ls -a"

Sekarang setiap ketik ls, otomatis akan menampilkan file tersembunyi juga.

üìã Melihat Alias yang Ada

Mau tahu semua alias aktif di terminal? Cukup ketik:

alias

Atau lebih lengkap:

alias -p

üîí Membuat Alias Permanen

Alias di terminal itu sementara. Biar permanen, tambahin ke file ~/.bashrc atau ~/.bash_aliases:

alias up='sudo apt update && sudo apt upgrade -y && sudo apt autoremove -y && sudo apt dist-upgrade -y'

‚ö†Ô∏è Hal yang Perlu Diperhatikan

  • Alias itu case-sensitive (huruf besar kecil beda).
  • Jangan sampai menimpa perintah penting Linux tanpa sadar.
  • Pakai type <nama> buat cek apakah nama alias sudah dipakai sistem.

‚ùå Menghapus Alias

Untuk hapus alias sementara:

unalias ls

Hapus semua alias sekaligus:

unalias -a

Kalau aliasnya di .bashrc, hapus barisnya lalu restart terminal.

üõ†Ô∏è Alias Lanjutan

  • Alias bisa buat shortcut script atau perintah panjang.
  • Bisa juga untuk mempercepat workflow harian.

Contoh alias unik:

alias cd..="cd .."
alias edit=nano

üö™ Escaping Alias

Kadang alias suka ganggu, misal grep sudah di-alias otomatis nambah parameter. Biar jalanin perintah asli, pakai backslash:

\grep bash

üéØ Kesimpulan

  • Alias itu fitur kecil yang sangat membantu.
  • Bisa bikin kerja lebih cepat & efisien.
  • Bisa dibuat permanen, dihapus, atau diabaikan kapan aja.

Kalau kamu sering ngoprek terminal, alias bakal jadi sahabat terbaikmu. Selamat berkreasi dengan alias!
