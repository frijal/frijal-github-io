Memilih Desktop Environment untuk Linux

Panduan santai memilih DE yang pas buat kebutuhan kamu ✨

Apa itu Desktop Environment?

Desktop Environment (DE) adalah antarmuka grafis (GUI) yang digunakan pengguna untuk berinteraksi dengan sistem operasi Linux. Dengan DE, kita bisa membuat folder, membuka aplikasi, dan mengatur sistem melalui klik mouse, bukan hanya lewat Command Line Interface (CLI).

Cara Memulai Desktop Environment Baru

  1. Setelah menginstal DE, logout dari sistem.
  2. Pada layar login, pilih ikon roda gigi (atau tombol serupa).
  3. Pilih DE yang ingin digunakan.
  4. Masukkan password, lalu login kembali.

Catatan: Lokasi tombol pemilih DE bisa berbeda tergantung distro (Ubuntu, Fedora, ArchLinux).

Beberapa Desktop Environment Populer

1. Budgie

  • Ringan dan efisien.
  • Banyak distro sudah menyediakannya secara bawaan.
  • Instalasi:
    • Ubuntu: sudo apt install ubuntu-budgie-desktop
    • Fedora: sudo dnf install budgie-desktop
    • ArchLinux: sudo pacman -S budgie-desktop

2. Cinnamon

  • Mirip Windows 10, cocok untuk transisi dari Windows.
  • Instalasi:
    • Ubuntu: sudo apt install cinnamon-desktop-environment
    • Fedora: sudo yum install cinnamon cinnamon-desktop
    • ArchLinux: sudo pacman -S cinnamon nemo-fileroller

3. Enlightenment

  • Ringan sekaligus indah secara visual.
  • Instalasi:
    • Ubuntu: sudo apt install enlightenment
    • Fedora: sudo yum install enlightenment
    • ArchLinux: sudo pacman -S enlightenment

4. LXDE

  • Sangat ringan, ideal untuk komputer lama.
  • Instalasi:
    • Ubuntu: sudo apt install lxde
    • Fedora: sudo yum groupinstall lxde-desktop
    • ArchLinux: sudo pacman -S lxde

5. LXQt

  • Versi modern LXDE berbasis Qt.
  • Lebih cepat dan tetap ringan.
  • Instalasi:
    • Ubuntu: sudo apt install lxqt
    • Fedora: sudo yum install @lxqt-desktop
    • ArchLinux: sudo pacman -S lxqt

6. MATE

  • Fork dari GNOME 2, stabil dan sederhana.
  • Instalasi:
    • Ubuntu: sudo apt install ubuntu-mate-desktop
    • Fedora: sudo yum install @mate-desktop
    • ArchLinux: sudo pacman -S mate mate-extra

Kesimpulan

Linux benar-benar modular dan fleksibel. Dengan memilih Desktop Environment yang sesuai, kita bisa membangun pengalaman Linux yang sesuai kebutuhan—apakah ringan, mirip Windows, atau penuh efek visual.
