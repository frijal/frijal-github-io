Membuat Bootable Linux Menggunakan USB Flash

Langkah 1: Download ISO Linux

Pertama, unduh file ISO dari distribusi Linux yang diinginkan.

Langkah 2: Pilih USB Drive

Colokkan USB flash drive minimal 4GB dan pastikan data penting telah di-backup.

Langkah 3: Gunakan Software Bootable

Gunakan software seperti Rufus (Windows) atau Etcher (Linux/Mac) untuk membuat USB bootable.

Langkah 4: Buat Bootable USB


# Contoh menggunakan dd di Linux
sudo dd if=linux.iso of=/dev/sdX bs=4M status=progress oflag=sync
# Ganti /dev/sdX sesuai USB drive Anda

    Langkah 5: Boot dari USB

    Restart komputer dan pilih USB sebagai boot device pada BIOS/UEFI.
