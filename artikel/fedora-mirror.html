Mengaktifkan plugin mirror tercepat untuk Fedora (DNF)

Penjelasan

Fedora menggunakan dnf sebagai package manager. Untuk mempercepat update dan instalasi, kita bisa mengaktifkan plugin fastestmirror yang otomatis memilih mirror tercepat. Skrip berikut akan mengaktifkan fitur tersebut dan membersihkan cache DNF.

‚ö†Ô∏è Jalankan dengan sudo karena skrip ini mengubah konfigurasi /etc/dnf/dnf.conf.

Kode Skrip

      
#!/bin/bash
# fedora-mirror.sh (aktifkan mirror tercepat Fedora dengan DNF)

set -e

echo "[INFO] Install plugin DNF..."
sudo dnf install -y dnf-plugins-core

echo "[INFO] Backup konfigurasi lama..."
sudo cp /etc/dnf/dnf.conf /etc/dnf/dnf.conf.bak.$(date +%F_%T)

echo "[INFO] Mengaktifkan fastestmirror di /etc/dnf/dnf.conf..."
if ! grep -q "fastestmirror=True" /etc/dnf/dnf.conf; then
  echo "fastestmirror=True" | sudo tee -a /etc/dnf/dnf.conf
fi

echo "[INFO] Membersihkan cache DNF..."
sudo dnf clean all

echo "[DONE] Fastestmirror berhasil diaktifkan untuk Fedora!"
