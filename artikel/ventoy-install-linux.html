💽 Cara Instal dan Menggunakan Ventoy 1.1.04 untuk Bootable USB di Linux

Sumber asli: Linux.org

🔎 Apa itu Ventoy?

Ventoy adalah tool open-source yang bisa bikin flashdisk jadi multi-boot USB. Instal sekali, lalu tinggal copy-paste file ISO, WIM, IMG, VHD, EFI ke USB. Saat boot, Ventoy kasih menu buat pilih OS. Support BIOS & UEFI, bahkan Secure Boot. Bisa juga persistence (nyimpen perubahan OS live). Praktis banget buat installer, tester, atau koleksi OS portable!

  • Instal Ventoy sekali aja
  • Cukup copy-paste file ISO/WIM/IMG/VHD/EFI ke USB
  • Menu pilih OS saat booting
  • Dukungan BIOS & UEFI, Secure Boot
  • Bisa persistence (save data di OS live)

📥 Download Ventoy

  1. Buka halaman download Ventoy
  2. Pilih file Linux: ventoy-1.1.04-linux.tar.gz
  3. Cek integritas file (penting!):
sha256sum -c SHA256SUM

Kalau hasilnya OK berarti file valid & aman dipakai!

📂 Ekstrak dan Instal Ventoy

Ekstrak file lalu install ke USB (ganti X dengan device, contoh sdb):

tar xvf ventoy-1.1.04-linux.tar.gz
cd ventoy-1.1.04
sudo sh Ventoy2Disk.sh -i /dev/sdX

👉 Pakai lsblk buat cek device USB yang benar, hati-hati jangan salah pilih disk!

🖥️ Mode GUI: VentoyWeb

Selain CLI, Ventoy juga punya mode GUI lewat browser, namanya VentoyWeb:

sudo sh VentoyWeb.sh

Buka http://127.0.0.1:24680 di browser → pilih USB → klik Install atau Update. Simpel dan friendly banget buat yang kurang nyaman di terminal.

📑 Jenis File yang Didukung

Ventoy bisa boot langsung dari banyak jenis file!

  • ISO
  • WIM
  • VHD / VHDX
  • IMG
  • EFI

💾 Persistence (Menyimpan Perubahan)

Biasanya OS Live itu kayak "reset" tiap boot. Tapi Ventoy bisa bikin file persistence, biar data atau setting tetap nyimpen:

sudo sh CreatePersistenceImg.sh -s 3000 -t ext4 -l casper-rw

Lalu hubungkan ISO ke file persistence pakai file JSON:

{
  "persistence": [
    {
      "image": "/linuxmint.iso",
      "backend": "/persistence.dat"
    }
  ]
}

Gampang kan? Sekarang live session bisa nyimpen dokumen, setting, app, dsb!

🛠️ Tool Tambahan Ventoy

  • VentoyVlnk.sh → Bikin link ke ISO lokal.
  • ExtendPersistenceImg.sh → Tambah ukuran file persistence.
  • VentoyPlugson.sh → GUI plugin web (theme, password, alias menu, dll).

🎯 Kesimpulan

Ventoy bikin hidup para tester dan kolektor OS jadi lebih mudah:

  • USB multi-boot, nggak perlu format ulang tiap kali.
  • Praktis buat coba-coba distro baru.
  • Dukung banyak format image.
  • Persistence, data nggak hilang tiap boot.
  • Pilih mode CLI atau GUI, sesuai gaya kamu!
