Cara Instal & Pakai Ventoy untuk Bootable USB di Linux

Fitur Persistence (Menyimpan Perubahan)

Kesimpulan

Langkah 1: Unduh dan Ekstrak Ventoy

Langkah 2: Instal Ventoy ke USB

Mode GUI: Menggunakan VentoyWeb

Ventoy adalah tool open-source yang memungkinkan Anda membuat flashdisk menjadi multi-boot USB. Cukup instal Ventoy sekali, lalu Anda bisa menyalin banyak file ISO, WIM, atau IMG ke dalamnya dan memilih OS mana yang akan di-boot dari menu.

Ganti /dev/sdX dengan nama device USB Anda. Anda akan diminta konfirmasi dua kali karena proses ini akan menghapus semua data di USB.

Gunakan perintah lsblk untuk menemukan nama device USB Anda (misalnya, /dev/sdb). Hati-hati, jangan salah pilih disk! Kemudian, jalankan skrip instalasi.

Ingin menyimpan perubahan pada Live OS? Ventoy mendukung fitur persistence. Anda bisa membuat file data terpisah yang akan digunakan oleh Live OS untuk menyimpan pengaturan, file, dan aplikasi yang diinstal.

Jika Anda lebih suka antarmuka grafis, Ventoy menyediakan mode web. Jalankan skrip berikut, lalu buka http://127.0.0.1:24680 di browser Anda.

Kunjungi halaman unduh Ventoy dan dapatkan file untuk Linux. Setelah itu, ekstrak arsipnya.

Ventoy adalah alat yang sangat praktis bagi siapa saja yang sering bekerja dengan banyak sistem operasi. Kemudahan penggunaan dan fleksibilitasnya menjadikannya pilihan utama untuk membuat multi-boot USB. Selamat mencoba!
