Memeriksa Kondisi Baterai di Linux

Pemeriksaan Menggunakan Command Line

Linux menyediakan beberapa cara untuk memeriksa kondisi baterai melalui terminal. Berikut beberapa perintah umum:

1. Menggunakan upower


# Menampilkan semua perangkat power
upower -e

# Menampilkan info baterai
upower -i /org/freedesktop/UPower/devices/battery_BAT0

2. Menggunakan acpi


# Menampilkan status baterai dan persentase
acpi -V

3. Menggunakan /sys/class/power_supply/


# Menampilkan kapasitas baterai
cat /sys/class/power_supply/BAT0/capacity

# Menampilkan status charging/discharging
cat /sys/class/power_supply/BAT0/status
