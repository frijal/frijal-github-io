Garuda Linux, distro berbasis Arch yang dikenal dengan tampilan visual memukau dan performa tinggi, merilis **ISO snapshot terbaru** untuk bulan September 2025. Ini bukan rilis mayor, melainkan pembaruan kumulatif yang mencakup semua paket terbaru, perbaikan bug krusial, dan peningkatan kompatibilitas perangkat keras sistem secara keseluruhan.

üîß Perubahan Utama

  • Mesa 25.2.2 ‚Äì Memperbaiki bug **logout acak** pada sesi **Plasma Wayland**, meningkatkan stabilitas grafis.
  • garuda-hardware-tool ‚Äì Peningkatan deteksi GPU, kini mendukung string PCI ID yang lebih panjang untuk kompatibilitas perangkat keras baru.
  • Dinonaktifkannya udevadm trigger ‚Äì Perubahan penting ini mencegah **crash sistem** yang dapat terjadi selama pembaruan file systemd/udev, menjamin proses pembaruan yang lebih aman.

Edisi andalan Garuda Linux, **Dr460nized (KDE Dragonized)**, juga mendapatkan pembaruan visual dan fungsional yang signifikan:

  • **Plasma 6.4.4** yang telah dikustomisasi secara mendalam.
  • Shell **Fish 4.0** sebagai shell default untuk pengalaman terminal yang lebih kaya fitur.
  • Kernel Linux **6.16** versi terbaru untuk performa dan dukungan perangkat keras optimal.
