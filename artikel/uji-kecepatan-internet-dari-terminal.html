Uji Kecepatan Internet dari Terminal

Panduan Santai untuk Para Penggemar Garis Perintah

Halo! Kamu pasti penasaran kan, gimana caranya cek kecepatan internet tanpa harus buka browser dan iklan yang bertebaran? Nah, ada cara keren banget buat para pengguna Linux atau Windows yang suka ngoprek, yaitu lewat terminal ! Yuk, kita bahas bareng gimana cara melakukannya. 🚀

---

Kenapa Pakai Terminal?

Mungkin kamu bertanya-tanya, "Kenapa sih harus repot-repot lewat terminal?" Jawabannya simpel:

  • Ringkas & Cepat: Tanpa iklan, tanpa embel-embel, hasilnya langsung muncul.
  • Bisa Otomatis: Kamu bisa menjadwalkan tes kecepatan secara otomatis, misalnya tiap jam, untuk memantau performa jaringan.
  • Keren!: Ya, harus diakui, melakukan sesuatu dari terminal itu ada sensasi "pro"-nya. 😎
---

Cara Pasang Speedtest-cli

Untuk melakukan tes ini, kita akan pakai sebuah alat bernama speedtest-cli. Alat ini tersedia di berbagai sistem operasi. Berikut cara pasangnya:

Di Linux (Berbasis Debian/Ubuntu)

Buka terminal dan ketik perintah ini:

sudo apt update
sudo apt install speedtest-cli

Di Linux (Berbasis Fedora/CentOS)

sudo dnf install speedtest-cli

Di Windows (PowerShell/CMD)

Kalau di Windows, kamu bisa pakai **Python** dan **pip**. Pastikan Python sudah terpasang, lalu jalankan perintah ini:

pip install speedtest-cli

Jika ada masalah, coba dengan pip3.

---

Mulai Uji Kecepatanmu!

Setelah terpasang, uji kecepatan internetmu semudah membalik telapak tangan. Cukup ketik perintah berikut di terminal:

speedtest

Lalu, tunggu sebentar. Hasilnya akan langsung ditampilkan dengan informasi lengkap: server yang digunakan, ping, kecepatan unduh (download), dan kecepatan unggah (upload).

Contoh Hasil

Retrieving speedtest.net configuration...
Testing from Telkomsel (118.96.11.233)...
Retrieving speedtest.net server list...
Selecting best server based on ping...
Hosted by Biznet (Jakarta) [0.47 km]: 1.348 ms
Testing download speed........................................
Download: 12.34 Mbit/s
Testing upload speed............................................
Upload: 5.67 Mbit/s

Klik gambar untuk melihat lebih jelas.

Fitur Lain yang Gak Kalah Keren

speedtest-cli punya beberapa trik tersembunyi, lho! Kamu bisa coba-coba perintah ini:

  • Pilih Server Tertentu: Kalau kamu mau tes ke server di lokasi lain, coba cari tahu daftar server yang tersedia. 
    speedtest --list
    Lalu, jalankan tes ke server yang kamu inginkan (misal, server ID 1234): 
    speedtest --server 1234
  • Tampilkan Hasil dalam Byte: Hasil defaultnya adalah Megabit (Mbit/s). Jika kamu lebih suka Megabyte (MB/s), gunakan opsi ini: 
    speedtest --bytes
  • Hasil Berbentuk JSON: Untuk yang mau ngolah data, ini berguna banget! 
    speedtest --json
