Merekam Bagian Layar Desktop dengan FFmpeg

Tutorial praktis untuk merekam area tertentu menggunakan perintah terminal

Bagi Anda yang sering membuat tutorial atau demo di komputer, merekam seluruh layar terkadang tidak efisien. Untungnya, dengan **FFmpeg**, kita bisa merekam hanya pada area tertentu di desktop. Ini sangat berguna untuk menghemat ruang penyimpanan dan memfokuskan rekaman pada konten yang penting saja.

Alat-alat yang Dibutuhkan

Sebelum memulai, pastikan Anda telah menyiapkan dua hal utama:

  1. FFmpeg: Pastikan aplikasi ini sudah terpasang di sistem operasi Anda. Anda bisa menginstalnya melalui terminal (contoh: `sudo apt install ffmpeg` di Ubuntu/Debian).
  2. Screen Ruler (Penggaris Layar): Alat bantu ini sangat penting untuk mendapatkan koordinat (posisi) dan ukuran area yang ingin Anda rekam dengan presisi. Ada banyak aplikasi gratis yang bisa Anda gunakan, salah satunya "Screen Ruler".

Perintah Dasar Merekam Sebagian Layar

Perintah berikut adalah "resep" utama untuk merekam sebagian area desktop. Perintah ini berlaku untuk sistem operasi berbasis X11 seperti sebagian besar distro Linux.

ffmpeg -f x11grab -s 700x500 -r 25 -i :0.0+1,1 -sameq googleplus.mpg

Penjelasan Detail Parameter Perintah:

  • ffmpeg: Perintah utama untuk menjalankan aplikasi FFmpeg.
  • -f x11grab: Ini adalah format input yang digunakan, yaitu `x11grab`. Perintah ini khusus untuk merekam tampilan di sistem X Window (umum di Linux).
  • -s 700x500: Parameter `s` (singkatan dari **size**) menentukan resolusi atau ukuran video yang akan direkam, yaitu 700 piksel lebar dan 500 piksel tinggi.
  • -r 25: Parameter `r` (singkatan dari **frame rate**) menentukan kecepatan tangkapan video, yaitu 25 frame per detik (fps). Nilai ini bisa Anda sesuaikan.
  • -i :0.0+1,1: Parameter `i` (singkatan dari **input**) ini sangat penting.
    • :0.0: Menunjukkan sumber input, yaitu desktop utama (layar monitor pertama) pada server X11.
    • +1,1: Ini adalah koordinat `x,y` dari titik awal (pojok kiri atas) area yang akan direkam. Dalam contoh ini, rekaman dimulai dari posisi 1 piksel dari kiri dan 1 piksel dari atas. Anda perlu mendapatkan nilai ini dari aplikasi penggaris layar.
  • -sameq: Parameter ini akan mencoba menjaga kualitas video agar sama dengan sumbernya, memastikan hasil rekaman tetap tajam.
  • googleplus.mpg: Nama file output. Anda bisa mengganti `googleplus.mpg` dengan nama file dan format video yang Anda inginkan (misal: `tutorial.mp4`).

Tips Tambahan

Untuk menghentikan proses perekaman, cukup tekan **`Ctrl + C`** di terminal Anda. FFmpeg akan otomatis menyimpan file video yang telah direkam.

Dengan memahami parameter di atas, Anda dapat dengan mudah menyesuaikan perintah sesuai kebutuhan Anda, mulai dari ukuran area, posisi, hingga kualitas video. Selamat mencoba!
