‚ö° 2. Upgrade Paralel dengan apt-fast

Biasanya update/upgrade download 1 file per waktu. Nah, dengan apt-fast kamu bisa unduh paket paralel, jadi upgrade makin kilat!

  • Install apt-fast:
    • sudo add-apt-repository ppa:apt-fast/stable
    • sudo apt install apt-fast
  • Pilih metode (apt-get/apt/aptitude) dan jumlah koneksi (default 5 aja sudah ngebut)
  • Mulai gunakan apt-fast buat update dan upgrade

üß† 3. Preload Aplikasi Favorit

Punya aplikasi yang sering banget dipake? Biar makin satset, install preload supaya aplikasi favorit kamu otomatis dimuat ke RAM.

  • Install: sudo apt install preload
  • Restart, lalu cek log-nya: cat /var/log/preload.log

üíæ 4. Optimasi SSD dengan fstrim

Kalau storage kamu SSD, aktifkan fstrim supaya ruang kosongnya makin efisien dan SSD lebih awet!

  • Cek status: sudo systemctl status fstrim.timer
  • Aktifkan kalau belum:
    • sudo systemctl enable fstrim.timer
    • sudo systemctl start fstrim.timer
  • Trim manual: sudo fstrim /media/$USER/ -v
