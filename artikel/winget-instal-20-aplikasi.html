    Winget itu apa sih?

     

    Winget bikin setup Windows jadi cepat, aman, dan otomatis. Panduan santai + skrip PowerShell untuk memasang 20 aplikasi sekaligus, lengkap dengan tips, ekspor/impor daftar, dan best practice.

     

    Biasanya, kalau habis install ulang Windows, kita harus unduh installer satu per satu: browser, pemutar musik, editor teks, dan seterusnya. Itu bisa makan waktu berjam-jam, plus ada risiko salah klik link palsu yang ternyata malware.

     

    Dengan Winget, cukup jalankan satu perintah. Contoh untuk 20 aplikasi populer:

     

    Instal manual 20 aplikasi bisa makan 2–3 jam. Dengan Winget, cukup 10–15 menit. Kamu juga bisa bikin skrip pribadi (batch/PowerShell) berisi daftar aplikasi favorit, jadi setiap kali setup ulang Windows tinggal jalankan skrip itu.

    Winget adalah manajer paket resmi dari Microsoft untuk Windows 10 dan 11, dirilis sejak 2020. Bisa dibilang ini “toko aplikasi” yang hidup di dalam Command Prompt atau PowerShell.
