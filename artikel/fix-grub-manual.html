Perbaikan Manual GRUB Bootloader di Debian/Ubuntu

Jika GRUB rusak atau hilang, kamu bisa memperbaikinya secara manual menggunakan Live USB. Cara ini lebih aman karena semua langkah diketik sendiri, jadi kamu tahu persis apa yang terjadi di sistem.

Langkah-Langkah

  1. Boot ke Live USB Debian/Ubuntu (gunakan opsi “Try Ubuntu/Debian without install”).
  2. Cari partisi root Linux: 
            
sudo fdisk -l
    Dari output, cari partisi Linux, misalnya /dev/sda1.
  3. Mount partisi root: 
            
sudo mount /dev/sda1 /mnt
    Jika ada partisi /boot terpisah, mount juga: 
            
sudo mount /dev/sda2 /mnt/boot
  4. Mount pseudo-filesystems: 
            
for i in /dev /proc /sys; do sudo mount --bind $i /mnt$i; done
  5. Masuk ke chroot: 
            
sudo chroot /mnt
  6. Reinstall GRUB ke disk (bukan partisi): 
            
grub-install /dev/sda
update-grub
  7. Keluar dari chroot dan unmount: 
            
exit
for i in /dev /proc /sys; do sudo umount /mnt$i; done
sudo umount /mnt
  8. Reboot sistem: 
            
sudo reboot
    Jika berhasil, GRUB akan muncul kembali.

Catatan: Jika sistem menggunakan UEFI, pastikan juga partisi EFI (biasanya /dev/sda1 dengan tipe FAT32) ikut di-mount ke /mnt/boot/efi sebelum menjalankan grub-install.
