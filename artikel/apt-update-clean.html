Berikut adalah contoh skrip shell .sh untuk memudahkan update dan membersihkan paket yang tidak terpakai di Debian/Ubuntu. Simpan dengan nama apt-update-clean.sh, lalu jalankan dengan:

‚ö†Ô∏è Catatan: Skrip ini membutuhkan akses sudo. Pastikan user kamu memiliki izin sudo sebelum menjalankan.

Skrip Lengkap

        
        #!/bin/bash
# Skrip otomatis update & clean APT
# Simpan sebagai: apt-update-clean.sh

echo "=== Memperbarui daftar paket ==="
sudo apt update -y

echo "=== Meng-upgrade paket terpasang ==="
sudo apt upgrade -y

echo "=== Menghapus paket yang tidak diperlukan ==="
sudo apt autoremove -y

echo "=== Membersihkan cache APT ==="
sudo apt autoclean -y
sudo apt clean -y

echo "=== Selesai! Sistem sudah diperbarui dan dibersihkan. ==="
Sitemap Home
    times;
