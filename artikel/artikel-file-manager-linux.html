🔎 Apa itu File Manager?

File manager di Linux itu kayak teman setia buat ngatur file: mulai dari bikin, mindah, hapus, sampai akses drive lokal maupun remote. Bisa juga pilih yang mana aja jadi default, sesuai selera dan workflow kamu!

  • Membuat, menyalin, memindahkan, menghapus file/folder
  • Mengakses drive lokal & remote (misal FTP, SMB, Google Drive)
  • Ganti file manager bawaan semudah ngopi, karena Linux itu modular banget!

🖥️ Cara Tahu File Manager Kamu

  • Buka folder Home, lalu cek menu Help → About.
  • Atau biar geeky, langsung aja di terminal:
    xdg-mime query default inode/directory
    Hasilnya misal nautilus.desktop atau nemo.desktop

📂 Pilihan File Manager Favorit

1. Caja (MATE Desktop)

  • Ringan, sederhana, bawaan desktop MATE.
  • Install:
    • Ubuntu: sudo apt install caja -y
    • Fedora: sudo yum install caja -y
    • Arch: sudo pacman -Syu caja

2. Dolphin (KDE Plasma)

  • Modern, fitur lengkap, integrasi KDE top.
  • Install:
    • Ubuntu: sudo apt install dolphin
    • Fedora: sudo yum install dolphin
    • Arch: sudo pacman -Syu dolphin

3. Konqueror

  • Seniornya Dolphin, bisa browsing juga lho!
  • Install:
    • Ubuntu: sudo apt install konqueror -y
    • Fedora: sudo yum install konqueror -y
    • Arch: sudo pacman -Syu konqueror

4. Nautilus (Files)

  • Bawaan GNOME, simpel dan stabil.
  • Install:
    • Ubuntu: sudo apt install ubuntu-desktop -y
    • Fedora: sudo yum install nautilus -y
    • Arch: sudo pacman -Syu nautilus

5. Nemo (Cinnamon Desktop)

  • Fork Nautilus, lebih fleksibel dan customizable.
  • Install:
    • Ubuntu: sudo apt install nemo -y
    • Fedora: sudo yum install nemo -y
    • Arch: sudo pacman -Syu nemo

6. Pantheon Files (Elementary OS)

  • Elegan dan simple banget khas Elementary OS.
  • Install:
    • Ubuntu: sudo add-apt-repository ppa:elementary-os/daily
      sudo apt install pantheon -y
    • Fedora: sudo yum install elementary-files -y
    • Arch: sudo pacman -Syu pantheon-files

7. PCManFM (LXDE)

  • Favorit netbook, cepat, dan irit resource.
  • Install:
    • Ubuntu: sudo apt install pcmanfm -y
    • Fedora: sudo yum install pcmanfm -y
    • Arch: sudo pacman -Syu pcmanfm

8. Thunar (XFCE)

  • Minimalis, cepat, dan stabil. Cocok buat laptop tua.
  • Install:
    • Ubuntu: sudo apt install thunar -y
    • Fedora: sudo yum install thunar -y
    • Arch: sudo pacman -Syu thunar

⚙️ Cara Ganti Default File Manager

Gampang banget! Tinggal pakai perintah ini:

xdg-mime default nama-aplikasi.desktop inode/directory

Contoh kalau mau pakai Nemo:

xdg-mime default nemo.desktop inode/directory

Biasanya .desktop file ada di /usr/share/applications/

🎯 Kesimpulan

Linux ngasih kebebasan penuh buat pilih file manager sesuai gaya dan workflow kamu, entah yang ringan, klasik, atau penuh fitur. Cobain aja satu-satu, pasti ketemu yang paling pas!
