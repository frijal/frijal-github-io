Jalankan Wizard Upgrade: Di Update Manager, buka menu Edit → "Upgrade to Linux Mint 22.2 Zara". Ikuti petunjuk di layar, centang konfirmasi, lalu klik "Apply". Proses ini akan mengunduh dan memasang paket baru. Reboot dan Verifikasi: Setelah proses selesai, restart komputer Anda. Untuk memastikan upgrade berhasil, buka "System Info" dari menu dan periksa apakah versi sistem sudah menunjukkan "Linux Mint 22.2 Zara". Update Semua Paket: Buka Update Manager (ikon perisai), klik "Refresh," lalu "Install Updates" untuk memastikan semua paket, terutama mint-upgrade-info, sudah versi terbaru. Restart jika diminta.
