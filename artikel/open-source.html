Open Source: Kisah Sukses Sumber Terbuka

Open Source (Sumber Terbuka) adalah sebuah kisah sukses yang menghiasi permulaan abad ini. Pada awal 2000an banyak yang meremehkan, namun filosofi Sumber Terbuka terbukti berhasil menarik perhatian banyak pihak.

Contoh Kesuksesan Open Source

WikiPedia

WikiPedia adalah situs ensiklopedi kolaboratif gratis yang diterjemahkan ke banyak bahasa dan menjadi acuan bagi banyak orang.

WordPress

WordPress mempermudah pengguna awam membuat artikel dan menampilkan konten ke Internet. Saat ini digunakan juga oleh situs besar, tidak hanya pemula.

Ubuntu

Ubuntu masih menjadi sistem operasi bebas terbuka yang paling banyak digunakan sejak 2004.

MySQL

MySQL adalah bagian dari stack LAMP (Linux - Apache - MySQL - PHP), digunakan di hampir seluruh server besar seperti Wikipedia dan Facebook.

Android

Android diterima baik oleh pengguna perangkat seluler karena kemudahan penggunaan, perawatan, dan fitur lengkap.
