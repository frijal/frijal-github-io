Memutar Berkas Multimedia di Fedora Linux

Secara default, Fedora Linux tidak menyertakan beberapa codec multimedia karena alasan lisensi. Namun, Anda tetap bisa memutar berbagai format audio maupun video dengan menambahkan paket-paket codec tertentu.

Paket yang Dibutuhkan

Untuk memutar file multimedia, Anda dapat menginstal beberapa paket berikut:

sudo dnf install gstreamer gstreamer-plugins-bad gstreamer-plugins-ugly gstreamer-plugins-good xine-lib-extras-freeworld

Penjelasan Singkat Paket

  • gstreamer ‚Üí framework multimedia utama.
  • gstreamer-plugins-good ‚Üí plugin berkualitas baik dan legal.
  • gstreamer-plugins-bad ‚Üí plugin tambahan dengan status masih diuji.
  • gstreamer-plugins-ugly ‚Üí plugin yang lisensinya terbatas namun dibutuhkan.
  • xine-lib-extras-freeworld ‚Üí pustaka tambahan agar player bisa memutar format populer.
