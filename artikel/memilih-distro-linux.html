üîé Apa itu Distribusi Linux?

Linux itu bukan satu produk milik satu perusahaan, lho! Kernel Linux bebas dipakai, jadi banyak komunitas & perusahaan bikin distro Linux versi mereka sendiri. Tiap distro ada tujuan & ciri khasnya:

  • Fokus ke server dengan trafik tinggi
  • Keamanan (firewall, jaringan sensitif)
  • Buat desktop harian
  • Distro ringan buat device kecil (router, smartphone, dll)

‚ùì Pertanyaan untuk Memilih Distro

Sebelum milih distro Linux, coba tanya diri sendiri:

  • Basis kodenya stabil?
  • Update paketnya gampang (apt-get, dnf, zypper)?
  • Upgrade ke versi mayor mudah?
  • Komunitasnya besar? Banyak software?
