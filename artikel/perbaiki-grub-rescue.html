Cara Memperbaiki GRUB Rescue Tanpa LiveCD/USB

Banyak pengguna Linux sering mencoba berbagai distro dan memasangnya di harddisk atau perangkat virtual. Namun, saat konfigurasi partisi salah, sering muncul error GRUB Rescue. Jangan panik, berikut langkah detail untuk memperbaikinya tanpa bantuan LiveCD/USB.

Langkah Memperbaiki GRUB Rescue

Ikuti langkah-langkah ini secara berurutan:

set prefix=(hdX,Z)/boot/grub
insmod (hdX,Z)/boot/grub/linux.mod
insmod part_msdos
insmod ext2
insmod gzio
set root=(hdX,Z)
linux /boot/vmlinuz-[versi-kernel] root=/dev/sdXY ro
initrd /boot/initrd.img-[versi-image]
boot

Keterangan:

  • (hdX,Z) adalah lokasi partisi boot. Misalnya /dev/sda1 = (hd0,msdos1).
  • /dev/sdXY: X = nama disk (a, b, c), Y = nomor partisi (1, 2, 3).
  • Gunakan tombol TAB untuk melihat daftar kernel dan initrd yang tersedia.
