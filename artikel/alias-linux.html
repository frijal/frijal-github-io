Menjalankan Perintah Asli (Escaping)

Bayangkan alias itu seperti nama panggilan untuk perintah terminal yang panjang. Ini adalah cara praktis untuk membuat shortcut, menghemat waktu, dan membuat kerja Anda di terminal jadi super efisien.

Setelah menambahkan baris di atas, jalankan source ~/.bashrc atau buka terminal baru agar perubahan aktif.

Alias yang dibuat langsung di terminal bersifat sementara. Agar permanen, tambahkan ke file konfigurasi shell Anda, biasanya di ~/.bashrc atau ~/.bash_aliases.

Ini akan menjalankan perintah ls yang sesungguhnya, mengabaikan alias apapun yang mungkin telah Anda buat.

Jika Anda perlu menjalankan perintah asli yang sudah tergantikan oleh alias, gunakan tanda backslash (\) di depannya.

Untuk menghapus alias pada sesi terminal saat ini, gunakan perintah unalias.

Untuk menghapusnya secara permanen, cukup hapus baris alias tersebut dari file ~/.bashrc Anda.
