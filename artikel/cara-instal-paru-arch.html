Mantap! Karena Paru itu adalah AUR Helper yang dibuat untuk ekosistem Arch Linux, Anda tidak perlu khawatir. Meskipun Anda tidak menggunakan CachyOS, Paru bisa diinstal dan bekerja sempurna di Arch Linux murni, Manjaro, EndeavourOS, dan turunan Arch lainnya.

Paru bekerja dengan memanfaatkan Pacman dan kemampuan sistem Anda untuk mengkompilasi paket dari AUR. Berikut panduan santai langkah demi langkahnya!

1. Instalasi Paru (Kompilasi dari AUR)

Karena Paru sendiri berasal dari AUR, kita harus mengkompilasinya. Ini membutuhkan *build tools* dasar di sistem Anda.

A. Pastikan Build Tools Tersedia

Langkah pertama adalah memastikan Anda memiliki paket yang diperlukan untuk kompilasi (mengubah kode sumber menjadi program yang bisa dijalankan). 

# Pastikan base-devel (untuk makepkg) dan git sudah terpasang
sudo pacman -S --needed base-devel git

B. Kloning dan Build Paru

Sekarang kita akan mengambil kode sumber Paru dari AUR menggunakan Git, dan mengkompilasinya menggunakan makepkg. 

# 1. Masuk ke folder sementara (direktori /tmp atau di home Anda)
cd /tmp

# 2. Kloning repository Paru dari AUR
git clone https://aur.archlinux.org/paru.git

# 3. Masuk ke folder yang baru dikloning
cd paru

# 4. Bangun (build) dan instal Paru
# Perintah -si akan menginstal dependensi yang dibutuhkan secara otomatis
makepkg -si

Anda mungkin akan diminta memasukkan password Anda beberapa kali selama proses ini. Setelah selesai, Paru sudah terinstal! Anda bisa menghapus folder /tmp/paru.

2. Cara Menggunakan Paru (Perintah Harian)

Selamat! Sekarang Anda bisa menggunakan Paru sebagai pengganti pacman untuk hampir semua tugas harian. Keuntungannya? Paru akan mengurus Pacman dan AUR sekaligus!

Update Sistem Penuh (Pacman + AUR)

Ini adalah perintah ajaib yang paling sering Anda gunakan:

paru

atau secara eksplisit:

paru -Syu

Instal Paket Baru (Dari Pacman/AUR)

Paru akan mencari di repositori resmi dulu, lalu di AUR. Lebih cerdas dari Pacman!

paru -S [nama_paket]

Mencari Paket di Kedua Repositori

Mencari paket di Repositori Resmi dan AUR:

paru -Ss [kata_kunci]

3. Keunggulan Paru di AUR

Saat Anda menggunakan Paru untuk menginstal paket dari AUR, Paru akan otomatis bertanya dan melakukan langkah-langkah yang biasanya Anda lakukan manual (seperti mengunduh Git, menjalankan makepkg, dan menginstal dependensi).

Saat proses instalasi paket dari AUR, Paru akan menyajikan langkah-langkah konfirmasi dan menampilkan PKGBUILD (skrip kompilasi) agar Anda dapat meninjaunya terlebih dahulu, menjadikannya lebih aman dan transparan!
