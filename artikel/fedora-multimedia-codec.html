Fedora: Biar Bisa Bernyanyi dan Menari!

Tutorial singkat agar Fedora kamu bisa memainkan berbagai format multimedia.

Seperti distro Linux lainnya, Fedora secara bawaan hanya bisa memutar berkas musik dengan format OGG. Nah, agar Fedora yang terpasang bisa memutar format lainnya seperti mp3, mp4, mkv, avi dan lainnya, kita perlu memasang paket tambahan.

Tahapan untuk Fedora 16

Untuk Fedora 16, ikuti langkah-langkah di bawah ini dengan membuka Terminal:

  1. Pertama, instal repositori RPM Fusion Free dan Non-free.
    2. sudo yum -y localinstall --nogpgcheck http://download1.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-stable.noarch.rpm http://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-stable.noarch.rpm
  2. Setelah itu, instal paket-paket multimedia yang diperlukan.
    3. sudo yum -y install gstreamer gstreamer-plugins-good gstreamer-plugins-bad gstreamer-plugins-ugly libdvdread libdvdnav lsdvd libdvdcss ffmpeg ffmpeg-libs gstreamer-ffmpeg libmatroska xvidcore xine-lib xine-lib-extras-freeworld xine-plugin
