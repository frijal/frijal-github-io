Linux dan Internet Protocol Television (IPTV)

IPTV (Internet Protocol Television) adalah teknologi yang memungkinkan Anda menonton siaran TV, film, atau serial melalui internet. Berbeda dengan layanan on-demand seperti Netflix, IPTV lebih sering digunakan untuk siaran langsung (live stream).

Aplikasi IPTV Populer: Hypnotix

Salah satu aplikasi IPTV yang paling populer di Linux adalah Hypnotix. Awalnya dikembangkan untuk Linux Mint, aplikasi ini juga bisa diinstal di Ubuntu dan distro lainnya. Tampilannya modern dan mudah digunakan.

Instalasi Hypnotix di Ubuntu

Cara termudah untuk menginstal Hypnotix di Ubuntu adalah melalui PPA untuk memastikan semua dependensi terpenuhi.

sudo add-apt-repository ppa:kelebek333/mint-tools
sudo apt update
sudo apt install hypnotix -y

Menambahkan Channel Baru

Anda dapat menambahkan lebih banyak channel dengan menggunakan file playlist M3U. Salah satu sumber populer adalah dari IPTV-org di GitHub.

  1. Unduh file M3U yang Anda inginkan.
  2. Di Hypnotix, buka menu Providers → Add a new provider.
  3. Pilih "Local M3U File" dan arahkan ke file yang baru saja Anda unduh, atau pilih "M3U URL" dan masukkan link playlist secara langsung.

Alternatif Pemutar IPTV Lainnya

Selain Hypnotix, ada beberapa pemutar media lain yang juga mendukung streaming IPTV:

  • VLC Media Player
  • MPV
  • SMPlayer
  • Celluloid
