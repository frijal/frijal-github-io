Sering bingung bedain <head> sama <header>? Yuk kita bahas dengan gaya santai, lengkap dengan contoh dan diagram.

üß© Apa itu <head>?

<head> adalah bagian dokumen HTML yang berisi metadata: judul halaman, deskripsi, link CSS, favicon, script, dan lain-lain. Bagian ini tidak terlihat langsung oleh pengguna, tapi penting untuk SEO dan browser.

üß© Apa itu <header>?

<header> adalah bagian atas halaman (di dalam <body>) yang terlihat oleh pengguna. Biasanya berisi logo, judul, menu navigasi, atau tagline.

‚ú® Analogi Gampang

  • <head> = "Otak" dokumen ‚Üí berisi metadata, tidak terlihat.
  • <header> = "Kepala" halaman ‚Üí bagian atas yang dilihat oleh pengguna.

üìê Diagram Struktur

<!DOCTYPE html>
<html>
  <head>
    <!-- Metadata seperti judul dan link CSS ada di sini -->
    <title>Judul Halaman</title>
  </head>
  <body>
    <header>
      <!-- Konten visual bagian atas seperti logo dan menu -->
      <h1>Selamat Datang di Situs Saya</h1>
    </header>
    <main>
      <!-- Isi utama halaman ada di sini -->
    </main>
    <footer>
      <!-- Konten bagian bawah halaman -->
    </footer>
  </body>
</html>
