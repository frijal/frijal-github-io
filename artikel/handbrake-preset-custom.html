🎬 Membuat Preset Kustom di HandBrake untuk Encoding Video Lebih Baik

Sumber asli: Linux.org

🔎 Apa itu HandBrake?

HandBrake itu aplikasi open-source buat transcoding video alias ngubah-ubah format video. Udah banyak preset bawaan, tapi biar hasil lebih pas sama selera dan device kita, mending bikin preset kustom sendiri. Enaknya, kamu bisa atur kualitas, ukuran file, dan performa sesuai hardware yang dipunya. Simpel tapi powerful!

Gambar 1: Menu Presets di HandBrake

⚙️ Dukungan Hardware Encoding

HandBrake bisa pake akselerasi hardware, biar encode makin ngebut:

  • NVENC → Nvidia Encoder
  • QSV → Intel Quick Sync Video
  • VCN → AMD Video Core Next
  • MF → ARM Media Foundation

Tips: Pastikan driver GPU udah ready ya!

  • Nvidia: cek status pake nvidia-smi
  • Intel: install intel-gpu-tools lalu cek pake vainfo
  • AMD: download driver di sini

Gambar 2: Preset Hardware di HandBrake

📐 Mengatur Dimensi Video

Di tab Dimensions, kamu bebas pilih resolusi output sesuai kebutuhan:

  • 4320p (8K)
  • 2160p (4K)
  • 1080p (Blu-Ray)
  • 720p (DVD)
  • 576p (PAL)
  • 480p (NTSC)

Gambar 3: Tab Dimensions di HandBrake

🎥 Mengatur Encoder Video

Di tab Video, kamu bisa pilih encoder sesuai tujuan:

  • H.265 (x265) → Lebih efisien, file lebih kecil!
  • Pilih 10-bit atau 12-bit kalau video HDR.

Gambar 4: Tab Video dengan opsi encoder

🔊 Mengatur Audio

Tab Audio buat atur:

  • Pilih track audio favorit yang mau dipertahanin.
  • Tentukan bahasa default.
  • Pakai Passthru (copy codec asli) atau fallback encoder kalo perlu.

Gambar 5: Tab Audio di HandBrake

💬 Mengatur Subtitle

Tab Subtitles buat atur opsi subtitle:

  • Add Closed Captions kalau ada subtitle bawaan.
  • Add Foreign Audio Scan buat deteksi subtitle paksa (misal terjemahan papan teks).
  • Burn-in Behavior biar subtitle dibakar permanen ke video.

Gambar 6: Tab Subtitles di HandBrake

💾 Menyimpan Preset Kustom

Udah beres setting? Klik Presets → Save As. Kamu bisa:

  • Beri nama preset biar gampang dicari.
  • Pilih kategori sesuai selera.
  • Bisa dijadiin default preset juga!

Gambar 7: Dialog Save Preset di HandBrake

🎯 Kesimpulan

  • Maksimalkan hardware GPU untuk encoding lebih cepat.
  • Atur resolusi, codec, audio, subtitle, semua sesuai kebutuhanmu.
  • Simpan preset, nggak perlu setting ulang tiap encode.

Workflow encoding videomu jadi lebih efisien, konsisten, dan pastinya sesuai selera!
