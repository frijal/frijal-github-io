Apa Bedanya target="_blank" dengan target=""?

Belajar HTML dengan santai + demo interaktif

Dalam HTML, atribut target pada tag <a> dipakai untuk mengatur bagaimana link dibuka. Biasanya ada dua yang sering membingungkan:

  • target="_blank": selalu buka link di tab/jendela baru.
  • target="" (kosong): sama saja dengan tidak menulis target, link dibuka di tab sekarang.

Jadi kalau kamu menulis target="", itu sebenarnya tidak ada efek khusus. Browser menganggapnya sama dengan default.
