Mengisi waktu luang: Desktop BlankOn + Compiz, Gnome-Do, xfishtank, Epiphany & Shiretoko

Coba-coba santai yang akhirnya jadi tampilan desktop menyenangkan. Ditulis ringan, ditambah panduan singkat, galeri, dan video rekam layar pakai GTK-RecordMyDesktop.

BlankOn Compiz Gnome-Do xfishtank Nautilus Epiphany Shiretoko GTK-RecordMyDesktop
Ringkasan kegiatan

Ini murni eksperimen untuk bersenang-senang dengan desktop: menggabungkan BlankOn, efek Compiz (animasi & jendela), peluncur aplikasi Gnome-Do, screensaver ikan xfishtank, dua browser (Epiphany & Shiretoko), dan pengelola berkas Nautilus. Seluruh aktivitas desktop direkam menggunakan GTK-RecordMyDesktop lalu hasil .ogv diunggah ke YouTube.

Efek visual: Compiz untuk window animations, exposé, dan transisi halus yang bikin segar dilihat.
Produktivitas: Gnome-Do mempercepat peluncuran aplikasi dan aksi umum dengan shortcut.
Nuansa unik: xfishtank sebagai pemanis visual yang hidup.
Eksplorasi web: Epiphany dan Shiretoko dibuka bersamaan untuk uji perilaku peramban.
Detail teknis cepat
  • Distro: BlankOn (basis Debian)
  • WM/Kompositor: Compiz
  • Peluncur: Gnome-Do
  • Visual: xfishtank
  • File manager: Nautilus
  • Browser: Epiphany & Shiretoko
  • Rekam layar: GTK-RecordMyDesktop (output .ogv)

Catatan: paket & nama repositori dapat berubah sesuai rilis. Sesuaikan dengan manajer paket Anda saat ini.

Langkah ringkas dan tips
  1. Aktifkan Compiz: pasang paket, set sebagai kompositor, lalu pilih efek secukupnya (hindari berlebihan agar ringan).
  2. Pasang Gnome-Do: atur shortcut peluncuran dan plugin sesuai aplikasi favorit.
  3. Tambahkan xfishtank: jalankan saat idle atau sebagai screensaver ringan.
  4. Siapkan Nautilus: urutkan tampilan folder sesuai alur kerja.
  5. Buka 2 browser: uji situs yang sama di Epiphany dan Shiretoko untuk banding kinerja.
  6. Rekam layar .ogv: pakai GTK-RecordMyDesktop, pilih area/layar penuh, simpan, lalu unggah ke YouTube.

Contoh perintah instalasi (sesuaikan distro/versi)

# Debian/Ubuntu/BlankOn style (sesuaikan nama paket di rilis Anda)
sudo apt update
sudo apt install compiz compiz-plugins gnome-do xfishtank nautilus recordmydesktop

# Menjalankan perekaman dasar (terminal)
recordmydesktop --on-the-fly-encoding --fps 25 --full-shots --v_quality 63 --s_quality 10

# Menghentikan perekaman: tekan Ctrl+C di terminal, file OGV tersimpan di direktori kerja

Nama paket dapat berbeda. Gunakan apt search atau manajer paket setempat.
