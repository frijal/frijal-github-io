Kenapa ini aman: Kamu memindahkan data penting dulu, melepas kaitan partisi agar tak terkunci, baru menghapus. Terakhir, GRUB dipasang ulang agar sistem bisa boot normal.

Peringatan: Operasi partisi berisiko. Backup data penting ke partisi Linux/drive eksternal terlebih dahulu. Baca setiap langkah pelan, jangan terburu-buru.

Langkah-langkah rapi

Ikuti urutan ini agar aman. Kita lakukan langsung dari Linux yang sudah terpasang, tanpa LiveCD/USB.

1) Pindahkan data penting ke partisi Linux

Salin dokumen, foto, proyek, dan file penting dari partisi Windows ke partisi Linux atau ke drive eksternal. Pastikan space cukup.

2) Pasang GParted (kalau belum ada)

sudo apt-get update
sudo apt-get install gparted

3) Jalankan GParted

sudo gparted

Pilih disk yang berisi Windows (misal /dev/sda). Identifikasi partisi Windows (umumnya NTFS, label ‚ÄúOS‚Äù, ‚ÄúWindows‚Äù, atau ‚ÄúSystem Reserved‚Äù).

4) Lepas kaitan partisi Windows

Klik kanan partisi Windows ‚Üí Unmount. Kalau tombol Unmount abu-abu, pastikan partisi tidak sedang dibuka di file manager.

5) Hapus partisi Windows

Klik kanan partisi Windows ‚Üí Delete. Bila ada beberapa (misal system reserved/boot), hapus semuanya yang terkait Windows. Tekan Apply untuk menerapkan.

6) Gunakan ruangnya untuk apa saja

Kamu bisa memperbesar partisi root/home Linux, atau membuat partisi data baru (ext4). Terapkan perubahan dan tunggu sampai selesai.

7) Pasang ulang GRUB agar booting mulus

# Deteksi sistem dan generate menu
sudo update-grub2

# Pasang GRUB ke disk utama (ganti sda bila disk kamu berbeda)
sudo grub-install /dev/sda

# Perbarui initramfs jika diperlukan
sudo update-initramfs -u

Setelah restart: Sistem langsung boot ke Linux. Menu Windows akan hilang. Semua ruang bekas Windows kini milikmu ‚Äî entah untuk /home yang lega atau partisi data khusus.
