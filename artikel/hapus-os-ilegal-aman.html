    1. Pindahkan Data Penting

    2. Pasang dan Jalankan GParted

    5. Pasang Ulang GRUB

      Panduan ini menunjukkan cara menghapus sistem operasi ilegal (seperti Windows bajakan) dari komputer Anda dengan aman, langsung dari Linux yang sudah terpasang, tanpa memerlukan Live CD/USB.

      Ganti /dev/sda jika disk utama Anda memiliki nama yang berbeda.

      Jika belum terinstal, pasang GParted, lalu jalankan dengan hak akses root.

      Langkah terakhir adalah memperbarui dan memasang ulang bootloader GRUB agar komputer Anda bisa booting langsung ke Linux tanpa error.

      Salin semua file penting dari partisi Windows ke partisi Linux atau ke drive eksternal.

      Setelah partisi Windows dihapus, Anda akan memiliki ruang kosong (unallocated). Anda bisa menggunakannya untuk memperbesar partisi Linux yang ada atau membuat partisi baru untuk data.

      Setelah restart, sistem Anda akan bersih, legal, dan boot langsung ke Linux. Ruang disk yang sebelumnya digunakan oleh OS ilegal kini menjadi milik Anda sepenuhnya.
