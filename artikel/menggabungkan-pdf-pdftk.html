Menggabungkan Berkas PDF dengan PDFtk: Solusi Cepat dan Efisien

Sering kali kita bekerja dengan banyak berkas PDF yang terpisah, seperti laporan bulanan, bab-bab buku, atau dokumen pendukung. Menggabungkan berkas-berkas ini menjadi satu file tunggal membuat manajemen lebih praktis, mudah dibagikan, dan hemat ruang penyimpanan.

1. Memastikan PDFtk Terpasang

PDFtk adalah toolkit PDF berbasis command line yang ringan dan cepat. Untuk memasangnya di Linux:

sudo apt-get install pdftk

Untuk Windows atau OS lain, unduh installer dari situs resmi PDFtk.

2. Menyiapkan Berkas PDF

  • Kumpulkan semua berkas PDF dalam satu folder.
  • Gunakan nama file sederhana tanpa spasi (misal: laporan-jan.pdf, laporan-feb.pdf).

3. Perintah Menggabungkan PDF

Buka terminal, lalu jalankan:

pdftk berkas1.pdf berkas2.pdf berkas3.pdf output berkas-gabungan.pdf

Penjelasan:

  • pdftk: menjalankan program.
  • berkas1.pdf berkas2.pdf ...: daftar file PDF yang digabungkan.
  • output: kata kunci hasil.
  • berkas-gabungan.pdf: nama file hasil gabungan.

Contoh:

pdftk Bab1.pdf Bab2.pdf Bab3.pdf output Buku-Lengkap.pdf

Keunggulan PDFtk

  • Cepat: jauh lebih singkat dibanding aplikasi GUI.
  • Ringan: hemat resource sistem.
  • Otomatisasi: mudah diintegrasikan ke skrip shell.
