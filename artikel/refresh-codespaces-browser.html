üîÑ Cara Rebuild Codespace dari Browser

  1. Buka repositori Anda di GitHub, lalu klik tab **Codespaces** ‚Üí pilih codespace yang mau dibuka.
  2. Codespace akan terbuka di browser dengan tampilan mirip VS Code.
  3. Tekan Ctrl+Shift+P (Windows/Linux) atau Cmd+Shift+P (Mac) untuk membuka *Command Palette*.
  4. Ketik `Rebuild` ‚Üí pilih salah satu:
    • Codespaces: Rebuild Container ‚Üí rebuild cepat dengan cache.
    • Codespaces: Full Rebuild Container ‚Üí rebuild total, semua cache Docker dibersihkan dan container dibangun ulang dari nol.
  5. Tunggu proses selesai. Semua file kode Anda tetap aman, hanya *environment* (dev container) yang di-refresh.

‚ú® Perbedaan Rebuild vs Full Rebuild

Mode Apa yang dilakukan Kapan dipakai
Rebuild Menggunakan cache image lama untuk mempercepat build. Kalau hanya ada perubahan kecil di devcontainer.json atau ingin environment segar tanpa buang waktu lama.
Full Rebuild Hapus semua cache image, download ulang base image, jalankan semua setup dari awal. Kalau environment error, dependency rusak, atau ingin benar‚Äëbenar fresh.

‚ö° Tips

  • Kalau cuma ingin "restart" service (misalnya Node, Python, server dev), cukup jalankan ulang service itu di terminal, tidak perlu rebuild penuh.
  • Kalau sering rebuild, pilih rebuild biasa dulu. Full rebuild hanya kalau masalah tidak hilang.
