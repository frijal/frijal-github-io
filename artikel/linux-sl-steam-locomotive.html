Mengenal Paket SL di Linux

Salah ketik jadi hiburan: dari ls ke sl

Siapa sangka, salah ketik perintah di terminal Linux bisa menjadi hiburan tersendiri. Saat bermaksud mengetik **ls** untuk melihat isi direktori, jari bisa saja terpeleset menjadi **sl**. Biasanya akan muncul pesan: 

The program 'sl' is currently not installed.
You can install it by typing: sudo apt install sl

Karena penasaran, coba saja ikuti saran tersebut dengan memasang paket SL (yang merupakan singkatan dari **Steam Locomotive**â€”Kereta Uap!): 

sudo apt install sl

Setelah terpasang, coba jalankan perintah:

sl

Hasilnya? Terminal Anda menampilkan animasi **kereta api uap** yang sedang melintas!

Lucu bukan? Dari salah ketik justru menghadirkan "stasiun" hiburan di terminal Linux Anda. Anda bisa mencoba beberapa opsi keren lainnya, seperti sl -a (kereta api yang salah mengira dirinya pesawat) atau sl -F (kereta api terbang)!
