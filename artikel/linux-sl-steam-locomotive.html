Mengenal Paket SL di Linux

Salah ketik jadi hiburan: dari ls ke sl

Home Home

Siapa sangka, salah ketik perintah di terminal Linux bisa menjadi hiburan tersendiri. Saat bermaksud mengetik ls untuk melihat isi direktori, jari bisa saja terpeleset menjadi sl. Biasanya akan muncul pesan:

The program 'sl' is currently not installed.
You can install it by typing: sudo apt install sl

Karena penasaran, coba saja ikuti saran tersebut dengan memasang paket SL:

sudo apt install sl

Setelah terpasang, coba jalankan perintah:

sl
