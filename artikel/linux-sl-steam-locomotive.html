Mengenal Paket SL di Linux: Salah Ketik jadi Hiburan Terminal

Siapa sangka, salah ketik perintah di terminal Linux bisa menjadi hiburan tersendiri. Ini adalah cerita tentang bagaimana kesalahan ketik ls menjadi sl justru menghadirkan kejutan tak terduga.

Twitter Facebook WhatsApp

Saat bermaksud mengetik ls untuk melihat isi direktori, jari bisa saja terpeleset menjadi sl. Biasanya, terminal akan memberikan pesan bahwa perintah tersebut tidak ditemukan, sekaligus menyarankan cara instalasinya:

The program 'sl' is currently not installed.
You can install it by typing: sudo apt install sl

Karena penasaran, mari kita ikuti saran tersebut. Paket **SL** adalah singkatan dari **Steam Locomotive** (Kereta Uap!).

sudo apt install sl

Setelah terpasang, coba jalankan kembali perintah sl:

sl

Hasilnya? Terminal Anda akan menampilkan animasi kereta api uap yang sedang melintas! Sebuah "hukuman" yang menghibur bagi jari yang salah ketik.

Anda bisa mencoba beberapa opsi lain yang tidak kalah seru, seperti sl -a untuk melihat "kecelakaan" kecil, atau sl -F untuk kereta yang terbang. Selamat mencoba! 😉
