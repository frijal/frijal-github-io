Pilih mirror tercepat untuk Debian/Ubuntu dengan netselect-apt

Penjelasan

Skrip debian-mirror.sh ini akan membantu memilih mirror tercepat untuk Debian/Ubuntu menggunakan netselect-apt. Setelah dijalankan, sources.list lama akan dibackup otomatis, lalu diganti dengan mirror tercepat hasil deteksi.

‚ö†Ô∏è Jalankan dengan sudo karena akan mengubah /etc/apt/sources.list.

Kode Skrip

      
#!/bin/bash
# debian-mirror.sh (otomatis pilih mirror tercepat Debian/Ubuntu)

set -e

echo "[INFO] Update & install netselect-apt..."
sudo apt update
sudo apt install -y netselect-apt

echo "[INFO] Menjalankan netselect-apt untuk rilis stable..."
sudo netselect-apt stable -n

echo "[INFO] Backup sources.list lama..."
sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak.$(date +%F_%T)

echo "[INFO] Mengganti sources.list dengan hasil mirror tercepat..."
sudo mv sources.list /etc/apt/sources.list

echo "[INFO] Update daftar paket..."
sudo apt update

echo "[DONE] Mirror tercepat berhasil dipasang!"
