Mengubah Format Gambar Massal dengan ImageMagick

Jika Anda memiliki ratusan gambar yang harus diunggah ke website, ukurannya sering kali terlalu besar. Mengkonversi gambar satu per satu sangat membuang waktu. Dengan **ImageMagick**, kita bisa mengubah format gambar secara massal agar lebih ringan menggunakan perintah **`mogrify`**.

Perbedaan `mogrify` dan `convert`

Penting untuk dipahami bahwa dalam ImageMagick terdapat dua perintah utama untuk konversi:

  • **`convert`**: Digunakan untuk mengkonversi gambar, **membuat file output baru** dan membiarkan file asli tetap utuh.
  • **`mogrify`**: Digunakan untuk mengkonversi gambar, namun **akan menimpa (overwrite) file asli** dengan format baru. Sangat efisien untuk operasi massal, tetapi gunakan dengan hati-hati.

Contoh Perintah Massal (PNG ke JPG)

Perintah ini akan mengubah **semua file PNG** di direktori aktif menjadi format **JPG** dan menimpa file aslinya. Jika Anda ingin mengubah format lain, cukup ganti `*.png` dan `jpg`.

mogrify -format jpg *.png

Penjelasan Sintaks

  • mogrify: Perintah utama dari ImageMagick yang memodifikasi file **di tempat** (menimpa file asli).
  • -format jpg: Opsi ini menentukan format tujuan konversi, yaitu Joint Photographic Group (JPG).
  • *.png: Merupakan *wildcard* yang menargetkan semua file yang berakhiran `.png` di direktori saat ini.
