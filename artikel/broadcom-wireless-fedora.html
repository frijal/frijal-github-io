Perbaikan Broadcom Wireless di Fedora

Setelah melakukan update kernel Fedora, beberapa perangkat wireless Broadcom mungkin tidak berfungsi. Hal ini biasanya terjadi karena modul kernel bcma yang bertentangan dengan driver Broadcom resmi. Berikut langkah-langkah mudah untuk memperbaikinya:

Langkah Perbaikan

  1. Buka terminal sebagai root atau gunakan sudo -i.
  2. Salin dan jalankan perintah berikut:
    3. echo "blacklist bcma" >> /etc/modprobe.d/broadcom-wl-blacklist.conf
  3. Reboot komputer untuk menerapkan perubahan.
