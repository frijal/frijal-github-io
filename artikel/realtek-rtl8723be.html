Langkah cepat

Inti: kita clone repo driver lalu jalankan installer. Ini versi ringkas dari catatan aslinya—lebih rapi dan siap tempur.

# 1) Siapkan alat build
sudo apt update && sudo apt install -y build-essential git dkms linux-headers-$(uname -r)

# 2) Clone repo driver RTL8723BE yang sudah teruji
git clone https://github.com/tarunbatra/fixRTL8723BE
cd fixRTL8723BE

# 3) Jalankan installer
chmod +x install.sh
sudo ./install.sh

# 4) Reboot biar modul ke‑load
sudo reboot

Catatan: Kalau pakai distro berbasis rpm, sesuaikan paket (gcc, make, kernel-devel, dkms). Setelah reboot, cek sinyal di indikator Wi‑Fi dan jalankan iwconfig kalau mau verifikasi.

Troubleshooting singkat

  • Modul belum ke‑load: coba sudo modprobe rtl8723be. Kalau ada konflik, blacklist modul lama di /etc/modprobe.d/.
  • Power management kebablasan: matikan sementara dengan sudo iw dev wlan0 set power_save off. Untuk permanen, atur di NetworkManager.conf.
  • Kecepatan turun naik: uji channel router, jarak, dan interferensi. Pastikan antena internal tidak di‑disable oleh BIOS.
  • Update kernel berubah lagi: pastikan instalasi via DKMS supaya build ulang otomatis saat kernel update.
