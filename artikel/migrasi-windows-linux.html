Migrasi Windows ke GNU/Linux - Panduan Lengkap

Artikel ini diketik untuk memberikan petunjuk strategi migrasi M$ Windows ke GNU/Linux, khususnya aplikasi Office dan File Server.

Latar Belakang

Mengapa membuang ratusan juta rupiah untuk lisensi bila tersedia solusi gratis?

Perusahaan perlu sadar penghematan lisensi sangat penting. Microsoft Indonesia dan BSA gencar menagih perusahaan untuk lisensi produknya.

Lisensi M$ Office Basic sekitar $150/PC, OS WinXP $140/PC, dan CAL server bisa mencapai miliaran rupiah untuk perusahaan menengah.

Strategi Migrasi

  • Analisa penggunaan PC dan aplikasinya secara detail.
  • Identifikasi PC untuk fungsi administratif.
  • Identifikasi PC yang menjalankan aplikasi tertentu dan platformnya.
  • Analisa deployment coverage untuk seluruh PC.

Langkah Tambahan

  1. Tes kompatibilitas aplikasi di dosemu dan samba.
  2. Periksa penggunaan fitur spesifik M$ Office di dokumen user.
  3. Backup semua file sebelum migrasi.
  4. Bimbing user dalam menggunakan aplikasi baru.
  5. Berikan contoh langsung perbandingan M$ Office vs LibreOffice.

Demikian pengalaman strategi migrasi dari M$ Windows ke GNU/Linux.
