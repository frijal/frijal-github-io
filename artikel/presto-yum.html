YUM Presto: Hemat Bandwidth Saat Update

Panduan singkat & mudah untuk pengguna distro Linux berbasis YUM yang ingin update lebih cepat dengan menghemat kuota internet.

Inti sumber (diringkas): Banyak orang mengira Linux butuh usaha besar. Nyatanya, ada cara sederhana untuk mempermudah, salah satunya memakai YUM Presto agar update lebih cepat/hemat bandwidth. Contoh: paket 57 MB bisa jadi ~8,5 MB setelah kompresi/delta — cocok untuk koneksi terbatas.

Apa itu YUM Presto?

YUM Presto adalah plugin untuk manajer paket YUM yang memanfaatkan Delta RPM. Alih‑alih mengunduh paket lengkap setiap update, sistem hanya mengunduh perubahan (delta) dari versi lama ke versi baru. Efeknya, ukuran unduhan bisa jauh lebih kecil—hemat waktu dan kuota.

  • Hemat bandwidth: Unduhan sering kali menyusut drastis.
  • Lebih cepat: Waktu update berkurang, terutama pada koneksi lambat.
  • Terintegrasi: Bekerja lewat YUM (tampilan & alur tetap familiar).

Catatan: Pada distro modern berbasis DNF, fitur delta RPM umumnya sudah tersedia/terintegrasi. Namun bila Anda masih memakai YUM, plugin yum-presto adalah solusi klasik yang ringan.

Cara Pasang & Aktifkan

Buka terminal, lalu jalankan perintah berikut untuk memasang plugin: 

# Pasang YUM Presto (untuk distro yang masih menggunakan YUM)
sudo yum install yum-presto

# Lanjutkan update seperti biasa
sudo yum update -y

Setelah terpasang, YUM akan otomatis mencoba menggunakan delta RPM bila tersedia di repositori. Jika delta tidak tersedia, YUM akan mengunduh RPM penuh seperti biasa (jadi tetap aman).

Tips Praktis

  • Pastikan repo mendukung delta: Tidak semua mirror menyediakan delta RPM. Jika hasilnya tidak “mengecil”, coba ganti mirror.
  • Performa CPU vs hemat kuota: Rekonstruksi paket dari delta butuh CPU untuk “menyusun ulang”. Di mesin lama, proses mungkin sedikit lebih lama—tapi unduhannya tetap hemat.
  • Tambahkan plugin terkait: Misal yum-plugin-fastestmirror agar otomatis memilih mirror tercepat.

Contoh Output & Tambahan Perintah

Contoh berikut memperlihatkan pola umum. Output asli bisa berbeda tergantung distro & repositori.

# Memeriksa info plugin (opsional)
yum info yum-presto

# Mengaktifkan mirror tercepat (opsional)
sudo yum install yum-plugin-fastestmirror

# Menjalankan pembaruan
sudo yum update

# DNF (sistem modern) — referensi alternatif:
# sudo dnf update
# (Pada beberapa distro, delta RPM sudah terintegrasi)
