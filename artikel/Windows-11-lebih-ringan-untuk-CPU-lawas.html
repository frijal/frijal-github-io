Membuat Windows 11 Lebih Ringan untuk CPU Lawas

Windows 11 bisa terasa berat pada komputer dengan CPU lawas. Berikut beberapa langkah optimasi agar performa meningkat.

Langkah-langkah Optimasi

  • Matikan efek visual & animasi yang tidak perlu.
  • Nonaktifkan startup program yang tidak penting.
  • Gunakan tema ringan (misal tema Windows klasik).
  • Nonaktifkan layanan background yang tidak diperlukan.
  • Gunakan Storage Sense untuk mengelola cache & temporary files.
  • Update driver secara berkala.
  • Gunakan antivirus ringan atau built-in Windows Defender.

Contoh Perintah / Script PowerShell Ringan


# Matikan animasi dan efek transparan
Set-ItemProperty -Path "HKCU:\Control Panel\Desktop" -Name "UserPreferencesMask" -Type Binary -Value 90,12,03,80,10,00,00,00

# Nonaktifkan startup program tertentu
Get-CimInstance Win32_StartupCommand | Where-Object { $_.Command -like "*nama_program*" } | Remove-CimInstance

# Matikan layanan yang tidak perlu
Stop-Service "Diagsvc" -Force
Set-Service "Diagsvc" -StartupType Disabled
times;
