• Boot dari USB: Nyalakan komputer Anda dari USB drive. Anda akan disambut oleh command line dengan user root@archiso.
  • Buat USB Bootable: Gunakan alat seperti Balena Etcher, Rufus, atau perintah dd untuk menulis file ISO ke USB drive.
  • Disk Configuration: Pilih disk target dan biarkan installer melakukan partisi otomatis jika Anda tidak yakin.
  • Kernel: Untuk pemula, pilihan default linux sudah sangat baik. Pilihan lain seperti linux-lts (stabilitas) atau linux-zen (gaming) tersedia untuk kebutuhan spesifik.
  • Profile: Pilih tipe instalasi (misalnya, Desktop) dan Desktop Environment (seperti GNOME, KDE, XFCE).
  • Unduh ISO Terbaru: Kunjungi situs resmi Arch Linux untuk mengunduh file ISO.
  • User Account: Buat akun pengguna non-root untuk penggunaan sehari-hari.
