Membagikan Aplikasi Debian dengan dpkg-repack

Banyak pengguna Debian ...

Langkah-langkah Menggunakan dpkg-repack

  1. Pastikan paket dpkg-repack sudah terpasang.
  2. Jika ingin menjalankan tanpa sudo, gunakan bantuan fakeroot: 
    sudo fakeroot -u dpkg-repack nama-paket
  3. Contoh: berbagi aplikasi Amarok: 
    dpkg-repack amarok
  4. File amarok.deb akan muncul di direktori saat ini.
